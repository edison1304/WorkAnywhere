WorkAnywhere

Multi-Agent Workspace for Claude Code. 서버에서 에이전트를 실행하고, 어디서든 세션을 이어가세요.

GitHub Features

Features

하나의 워크스페이스에서 여러 에이전트를 동시에 관리하고, 서버의 모든 데이터를 실시간으로 동기화합니다.

🤖

Multi-Agent Orchestration

Project → Phase → Task 계층으로 여러 Claude 에이전트를 동시에 실행하고 관리합니다. CPU-style 스케줄링으로 자원을 효율적으로 분배합니다.

🔄

Real-time Multi-Client Sync

NDJSON 이벤트 로그 + flock 원자적 시퀀스 기반 동기화. 여러 클라이언트가 동시에 작업해도 데이터가 일관성을 유지합니다.

📱

Mobile Vibe Coding

Android 앱에서 SSH 터널을 통해 동일한 세션에 접속. 채팅 UI로 에이전트와 대화하고, 퍼미션을 승인하고, 결과를 확인합니다.

🔒

Server-Centric Architecture

모든 데이터는 서버에 저장됩니다. 클라이언트는 SSH로 접속하여 데이터를 미러링합니다. 별도 데이터베이스가 필요 없습니다.

📊

Intent Lock & Alignment

에이전트의 작업 범위를 잠그고(Intent Lock), AI가 본목적에서 벗어나는지 실시간으로 감지합니다 (Alignment Score).

⏱

Timeline & Compaction

완료된 태스크를 3-bucket 구조(완료/우회/에러)로 자동 정리. 한 눈에 무슨 일이 있었는지 파악할 수 있습니다.

How It Works

데스크톱에서 시작하고, 모바일에서 이어가세요.

Connect

SSH로 원격 서버에 연결합니다. 로컬 머신에서도 동작합니다.

Organize

Project → Phase → Task 계층으로 작업을 구조화합니다.

Run Agents

각 Task에 프롬프트를 입력하고 Claude 에이전트를 실행합니다.

Monitor & Interact

실시간 로그, 퍼미션 승인, 메시지 전송. 데스크톱과 모바일 모두에서.
